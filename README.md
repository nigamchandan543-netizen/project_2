## 📌 Project Description
This project is a menu-driven Python program that demonstrates the use of loops, nested loops, control statements, and logical problem-solving.

The program allows users to:
- Generate a right-angled triangle pattern
- Analyze a range of numbers
- Identify odd and even numbers
- Calculate the sum of all numbers in a given range
- Exit the program safely

## 🎯 Objective
The objective of this project is to practice:
- for loops
- while loops
- range() function
- Nested loops
- break
- continue
- pass
- Input validation
- Menu-driven programming

## ✨ Features
- Right-angled triangle pattern generation
- Number range analysis
- Odd and even number checking
- Sum calculation
- Error handling for invalid input
- Exit message

## 🛠️ Technologies Used
- Python 3

## 📂 Program Flow
1. Display welcome message
2. Show menu options
3. Generate pattern
4. Analyze number range
5. Display odd/even numbers
6. Calculate total sum
7. Exit the program

## ⚠️ Assumptions
- The user enters valid integer values.
- The number of rows should be greater than 0.
- The ending number should be greater than or equal to the starting number.
